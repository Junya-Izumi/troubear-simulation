# 熊ったシミュレーション
## 概要
熊と遭遇した時の対応の仕方を学ぶ3択のクイズゲームです<br>
これは第3回全国商業高校Webアプリコンテストに提出した作品です。<br>
- コンテストのホームページ [全国商業高校Webアプリコンテスト](https://showebcon.com/)<br>
- 提出した作品のサイト   [熊ったシミュレーション](https://troubear-simulation.netlify.app/)
### 技術スタック
- フロントエンド: html / css / JavaScript (Vanilla)
### 作成期間
約2か月 部活動の時間で作成しました
